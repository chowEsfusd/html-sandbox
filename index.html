This is my website

This is meant to be a basic introduction to GitHub Pages. There will be unfamiliar parts — especially any HTML5-specific code — but it shouldn't be totally uncomfortable. Take note of what you find familiar and what you find unfamilar!

The main goal is to show you how GitHub pages work and enough of a foundation that you can upload an existing static website (HTML/CSS/images/etc.). Secondary goals include showing you what a well-organized project looks like, some "best practices" when writing HTML/CSS, and a bit of what's new with HTML5.

We want to teach you how to deploy this site to GitHub Pages. Deploying is the process of taking code on your personal computer and putting it somewhere on the internet so that other people can access it.

We've written a guide to deploying this site to GitHub pages, so go take a read!
